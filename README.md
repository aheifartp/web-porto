# Rafie Habibi - Personal Portfolio

A modern, interactive personal portfolio website showcasing my experience, skills, and projects in IoT, Networking, and Software Development. Originally built with vanilla HTML/CSS/JS, this project has been fully migrated to a component-driven architecture using **Next.js** and **React** for better performance, modularity, and scalability.

## 🚀 Live Demo
https://rafiehabibi.my.id

## 🛠 Tech Stack

This project leverages modern web technologies to deliver a fast and dynamic user experience:

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Theming**: `next-themes` (Seamless Dark / Light mode toggling)
- **Icons**: [Phosphor Icons](https://phosphoricons.com/)
- **Animations**: [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)

## ✨ Key Features

- **Component-Based Architecture**: Broken down into modular React components (`Hero`, `About`, `Experience`, `Skills`, `Projects`, etc.) for maintainability.
- **Dynamic Project Filtering**: Interactive tabs to filter projects by categories (IoT, Networking, ML/AI, Web) using React state.
- **Scroll-Spy Navigation**: The navbar automatically highlights the currently viewed section as you scroll down the page.
- **Dark Mode Support**: Fully integrated dark mode that respects system preferences and allows manual toggling.
- **Fully Responsive**: Optimized for seamless viewing on desktops, tablets, and mobile devices.

## 💻 Getting Started

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aheifartp/web-porto.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd web-porto
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.

## 📂 Project Structure

- `/src/app` - Contains the main Next.js App Router layout and entry page.
- `/src/components` - Reusable UI components (Navbar, Footer, Hero, etc.).
- `/public` - Static assets including images, PDFs, and vector graphics.
- `/legacy` - Backup of the original vanilla HTML/CSS/JS implementation.
